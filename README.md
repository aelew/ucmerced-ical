## Run Locally

Clone the project

```bash
git clone https://github.com/aelew/ucmerced-ical.git
```

Go to the project directory

```bash
cd ucmerced-ical
```

Install dependencies

```bash
npm i
```

Start development server

```bash
npm run dev
```
